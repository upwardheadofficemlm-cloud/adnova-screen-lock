Overview

AdNova Screen Lock is a professional Android application designed for digital signage devices. It provides secure screen locking capabilities with Firebase backend integration for remote management and monitoring.

Multiple Lock Types

Full screen, edge-based, and custom area locking options for flexible protection.

PIN Protection

Optional PIN-based unlock system for enhanced security and parental controls.

Firebase Integration

Real-time device management, configuration updates, and analytics through Firebase.

Kiosk Mode

Complete device lockdown with system button blocking for dedicated signage use.

Installation

1

Download the APK

Download the AdNova Screen Lock APK file from the official website or admin dashboard.

2

Enable Unknown Sources

Go to Settings → Security → Unknown Sources and enable installation from unknown sources.

3

Install the Application

Open the downloaded APK file and follow the installation prompts.

4

Grant Permissions

The app will request necessary permissions. Grant all permissions for full functionality.

Important: Ensure your device meets the minimum requirements: Android 10+, 2GB RAM, 32GB storage.

Initial Setup

1

Launch the Application

Open AdNova Screen Lock from your app drawer or home screen.

2

Grant Overlay Permission

When prompted, grant the "Display over other apps" permission. This is essential for the lock functionality.

3

Configure Basic Settings

Set your preferred lock type, unlock method, and other basic settings in the main interface.

4

Test the Lock

Activate the lock and test the unlock method to ensure everything works correctly.

Lock Types

Full Screen Lock

Locks the entire screen, preventing any touch input across the whole display.

Lock Type: Full Screen Coverage: 100% of screen area Use Case: Complete device protection

Edge Lock

Locks specific edges of the screen (left, right, top, bottom) while keeping the center area accessible.

Lock Type: Edge Lock Coverage: Selected edges only Use Case: Partial protection with center access

Custom Area Lock

Allows you to define custom rectangular areas to lock, providing precise control over protected zones.

Lock Type: Custom Area Coverage: User-defined rectangles Use Case: Specific area protection

Unlock Methods

Floating Unlock Button

A floating button that appears on screen for easy unlocking. Configurable position and behavior.

Double Tap

Double-tap the floating button to unlock the screen.

Tap Counter

Set the number of taps required on locked areas before the button appears.

Position Control

Choose from predefined positions or set custom coordinates for the button.

PIN Unlock

Optional PIN-based security for additional protection and parental controls.

1

2

Set PIN

Enter a 4-8 digit PIN and confirm it.

3

Test PIN Unlock

Lock the screen and test the PIN unlock functionality.

Troubleshooting

App won't lock the screen

Ensure the "Display over other apps" permission is granted. Go to Settings → Apps → AdNova Screen Lock → Permissions and enable "Display over other apps".

Floating button not appearing

Check the tap counter settings. The button may require multiple taps on locked areas before appearing. Also verify the button position settings.

PIN unlock not working

Verify that PIN protection is enabled in settings. If you forgot your PIN, you may need to reset the app data or contact your administrator.

Firebase connection issues

Check your internet connection and ensure the device can reach Firebase services. Verify that the Firebase configuration is correct.

App crashes on startup

Ensure your device meets the minimum requirements (Android 10+, 2GB RAM). Try clearing the app data and cache, then reconfigure the settings.

Frequently Asked Questions

Q: Can I use this app on tablets and phones?

A: Yes, AdNova Screen Lock works on both Android tablets and phones running Android 10 or higher.

Q: Is the app free to use?

A: The basic version is free. Premium features and enterprise management require a subscription.

Q: Can I manage multiple devices from one admin account?

A: Yes, the admin dashboard allows you to manage unlimited devices from a single account.

Q: What happens if I forget my PIN?

A: Contact your system administrator or reset the app data (this will require reconfiguration).

Q: Can the app work offline?

A: Basic locking functionality works offline, but remote management and analytics require an internet connection.

Q: Is my data secure?

A: Yes, all data is encrypted and stored securely using Firebase's enterprise-grade security.
